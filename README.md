# New York Times Article Scraper

## Overview
An app that scrapes articles from the New York Times website and lets users save articles and leave comments on the latest news.


## Technologies Used
- Node.js
- Express.js
- Handlebars.js
- MongoDB
- Mongoose
- npm packages
    - body-parser
    - express
    - express-handlebars
    - mongoose
    - cheerio
    - request
